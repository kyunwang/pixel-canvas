# Pixel Canvas

A teeny pixel art canvas with the dimensions of 30 x 30

![](src/assets/preview.png)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
